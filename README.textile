h2. 実行手順

h3. 1. Simple Build Tool 起動

<pre>
$ bin/sbt
</pre>

h3. 2. コンパイル

<pre>
> compile
</pre>

h3. 3. http://localhost:8080 に起動

<pre>
> container:start
</pre>

h3. 4. ソースコード修正後に挙動を確認したい場合

<pre>
> compile
> container:reload /
</pre>

h3. 5. Simple Build Tool 終了したい場合

<pre>
> exit
</pre>

h2. デプロイ手順

h3. 1. stage

<pre>
$ bin/sbt clean compile stage
</pre>

h3. 2. push

<pre>
$ git push heroku master
</pre>
